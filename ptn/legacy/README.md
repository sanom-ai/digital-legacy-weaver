# PTN Legacy Boundary

This path is reserved for PTN Legacy proprietary modules.

Public repository policy:

1. Do not commit proprietary `.ptn` policy packs to this public repository.
2. Keep only stubs/templates or documentation metadata in public.
3. Store real proprietary PTN legacy packs in private infrastructure/repositories.
4. Access, copy, and reuse of proprietary PTN legacy packs requires written permission.

Licensing:

1. Open-core code remains under `LICENSE` (MIT).
2. PTN Legacy proprietary assets are governed by `LICENSE-PTN`.
