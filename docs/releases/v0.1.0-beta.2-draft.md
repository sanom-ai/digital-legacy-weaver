## Private-first Profiles

1. `Confidential`: keeps only high-level outcome markers and avoids detailed requirement traces.
2. `Minimal`: recommended default for beta; keeps sanitized control-state only.
3. `Audit-heavy`: keeps sanitized evidence/owner references for deeper operational review.
