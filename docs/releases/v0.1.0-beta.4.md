# Digital Legacy Weaver v0.1.0-beta.4

This beta refresh removes the dead-end backend blocker and replaces it with a clearer startup path for first-time testers.

## Highlights

1. Builds without live Supabase configuration now open a guided landing screen instead of stopping at a raw configuration blocker.
2. Testers can choose `Open demo workspace` to explore the local UX, intent builder, canonical artifact flow, and runtime readiness journey immediately.
3. Setup guidance remains visible with the required `--dart-define` values for wiring a real backend later.
4. Validation baseline for this release pass is green:
   - `python -m pytest _support/tests`
   - `flutter analyze`
   - `flutter test`

## Downloads

1. Android APK
2. Windows ZIP

## Boundary

1. Digital Legacy Weaver remains a technical companion.
2. Demo mode is local UX only and does not replace a live backend or legal process.
3. This beta remains intended for controlled testing and validation.
