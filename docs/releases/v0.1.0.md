# Digital Legacy Weaver v0.1.0

Digital Legacy Weaver begins as a newly emerging platform for private-first digital legacy coordination.

## Release definition

`v0.1.0` establishes the first formal baseline for:

1. Policy-driven trigger orchestration
2. Secure unlock workflow with one-time access patterns
3. Partner-ready handoff posture with clear legal-boundary language
4. Operational safety controls and runbook-oriented reliability practice

## Foundation capabilities

1. Flutter app baseline for owner and beneficiary flows
2. Supabase-backed schema, migrations, and edge-function runtime
3. PTN policy parser/evaluator and policy-pack examples
4. CI quality gates and runtime workflow scaffolding
5. Security event logging, rate limits, and emergency safety controls

## Product posture

1. Technical coordination layer for digital legacy continuity
2. Private-first operations and auditability focus
3. Legal companion model, not legal authority replacement

## Licensing model

1. Open-core repository code and public examples: MIT
2. PTN legacy proprietary boundary and premium PTN assets: LICENSE-PTN

## Forward path

`v0.1.x` is the early growth line focused on:

1. reliability hardening in real operating environments
2. stronger policy-runtime control coverage
3. partner-ready delivery quality for production pathways
