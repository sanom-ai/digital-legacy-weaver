# Beta Note: Private-first Profiles

This beta introduces privacy presets that help users choose a runtime logging posture in plain language:

1. `Confidential`
2. `Minimal`
3. `Audit-heavy`

What changes:

1. Flutter onboarding and safety settings now expose preset-based privacy selection.
2. PTN policy documents can declare `privacy_profile` directly.
3. Runtime uses the stricter posture between active PTN policy and user private-first settings.

Boundary reminder:

1. Digital Legacy Weaver is a technical companion.
2. It helps coordinate secure delivery and recovery workflows.
3. It is not a legal will.
4. It is not a legal adjudication service.

Closed-beta communication guidance:
